# UI Mockup: Inventory tab — dusk village backdrop

An illustrated UI mockup screenshot of a premium mobile JRPG / gacha-banner game's inventory screen — anime-painted UI panel overlay on top of the frontier village hub at dusk. Inventory shows currency header + category filter + item grid + selected detail. 16:9 wide.

**FIGURE GUARD**: Character ref is stylization only. No character figure in this modal — items only.

## Composition

Layered: hub backdrop dimmed ~35% + frosted glass modal ~80% × ~85% centered.

### Top — Currency Header (~10%)

Horizontal bar with two currency displays:
- **Left**: Gold currency — stylized illustrated gold coin icon (warm yellow, painterly metallic) + numeric placeholder (large abstract glyph digits)
- **Right**: Echo currency — stylized illustrated crystal shard icon (cool teal-violet, glowing) + numeric placeholder

Header has subtle warm parchment frame, both currency icons painterly anime style.

### Left Side — Category Filter (~15% of modal width)

Vertical sidebar with category tabs (top to bottom):
- All (faceted glyph)
- Weapon (sword icon)
- Armor (chest plate icon)
- Accessory (amulet icon)
- Consumable (potion vial icon)
- Blueprint (scroll icon)

The Weapon tab is **currently selected** with a warm gold rim glow. Each tab has a small icon glyph + placeholder name (abstract glyphs). Active count badge on each tab (small numeric placeholder).

### Center Main — Item Grid (~55% of modal width)

A **6 × 4 grid of 24 item cards** (some empty slots faint). Each item card is a square tile:
- Center: stylized illustrated item icon (sword variants, painterly anime style)
- Bottom strip: tiny placeholder name + tier rune
- Top-right corner: rarity gem dot (epic-purple / rare-blue / common-grey)
- Top-left corner: small quantity number for stackable items (or empty for unique)

Visual variety: ~12 weapon icons (sword / spear / dagger / bow / staff / hammer variants), each clearly distinct. ~5 cards show "equipped" small badge in the corner. ~7 cards are empty slot outlines.

The currently selected item has a **warm gold rim glow** (one weapon stands out).

A small filter / sort toolbar at the top of the grid (sort by rarity / tier / new — small icon glyphs).

### Right Side — Selected Item Detail (~25% of modal width)

Vertical detail panel:
- Top: **large item icon** (painterly, more detailed than grid version)
- Middle: placeholder name strip + tier rune + rarity gem
- Below: 3-4 affix lines (each line: icon + placeholder + numeric)
- Bottom action buttons: **Equip** (warm gold) / **Sell** (cool grey) / **Compare** (deep navy)

A small "owned by" indicator showing which character (if equipped) has it — small portrait token + name placeholder.

**Workshop catalog.** The inventory feels organized but lived-in — a quartermaster's clear tables. Each item readable, currency at the top, action close at hand. Painterly anime UI tone consistent with other modals.

- Background dim ~35%, cool slate
- Modal: warm parchment frame with frosted-glass interior
- Currency header: warm gold + cool teal-violet (Echo) — distinct accents
- Selected item: warm gold rim glow

- Item icons: painterly anime, distinct colors per rarity
- Action buttons: warm gold (equip primary) / grey (sell) / navy (compare)

## What this is NOT

- NOT wireframe, NOT photoreal screenshot, NOT legible text
- NOT cluttered (grid spacious, sidebar clean)
