Thank you for purchasing Fantasy Worlds asset pack!

Content is by default configured for URP.

If your project is using BIRP or HDRP, use the included content packs in Fantasy Worlds - Forest folder. 
Double click the FW01_BIRP_Content or FW01_HDRP_Content then hit import - all shaders, materials and scenes will be converted into respective rendering pipeline.

More detailed documentation can be found here: https://docs.google.com/document/d/1e44qZu1rXauBi3lGeVRJUAfbHxlHw1xmMPV00Mxays0

If you encounter any issues with the pack, please contact us at support@triforge.eu.
Remember to include Unity Editor version, render pipeline version and your
purchase invoice number to speed up the support proces.

Lastly, if you like the asset, please consider leaving a review on the Asset Store! It helps a lot with exposure and any feedback helps us in making our assets even better!
