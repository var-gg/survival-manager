## 목적

세력 쌍별 오해, 실제 진실, 충돌축, encounter link를 한눈에 정리한다. 본 문서는 세력 갈등의 관계형 source of truth다. ADR-0024 reskin baseline 위에서 4 인간 세력(솔라룸/이리솔 부족/회상 결사/그물 결사)의 정치·문화·신앙 갈등으로 정의한다.

## 왜 별도 문서가 필요한가

world bible은 각 세력의 단독 정의를 소유하고, campaign arc는 chapter-level 진행을 소유한다. faction pair의 충돌 관계는 별도 매트릭스로 분리해야 중복이 줄어든다.

## 충돌 행렬 상세

### 솔라룸 vs 이리솔 부족 (`faction_solarum` × `faction_wolfpine_tribes`)

| 항목 | 내용 |
| --- | --- |
| 표면 갈등 | 국경 약탈과 보복 |
| 숨겨진 진실 | 둘 다 심장로 파편에 반응해 오판하고 있다 |
| encounter_link | `weakside_dive`, `protect_carry` |
| first_resolution_window | `chapter_1_site_2` |
| 솔라룸→이리솔 찌르기 | "너희가 영역을 지키지 못했으니 우리가 관리한다." 문명·질서의 우위를 전제로 이리솔 부족의 자율성을 부정. 단린의 경우 교리적 확신이 이 방향으로 작동 |
| 이리솔→솔라룸 찌르기 | "너희가 세운 문이잖아. 너희 손으로 닫았고, 너희 손으로 부쉈어." 솔라룸의 위선과 자기 파괴를 직설적으로 찌름. 이빨바람은 철·돌담·약탈의 물리적 증거를 감각으로 읽어서 찌름 |
| 금기어 | 이리솔 부족에게 "산골뜨기/짐승/야만인" (생활 양식·결사 부정). 솔라룸에게 "침략자" (실제로 솔라룸이 먼저 국경을 확장) |
| 증거 사물 | 씨족 토템 (짓밟힌 것), 국경문 철 파편, 회조 체내 파편, 토템의 진동 |
| 양보 비용 | 솔라룸: 국경 단독 통제 주권 양보, 영역 표식 파괴 중단. 이리솔 부족: 독자 보복 포기, 합의 없는 선제 습격 중단. 이빨바람 개인: 분열된 씨족 일부와 일시 단절(회수 가능 hook — 후속 외전에서 화해 복원 가능) |

**구체적 오해 에피소드**: 솔라룸 초소가 이리솔 부족 사냥대를 약탈자로 오인해 선제 사격한다. 이리솔 부족은 팩 토템이 짓밟힌 것에 대한 보복으로 초소를 습격한다. 양측 모두 '먼저 공격한 건 상대편'이라고 믿는다. 실제로는 심장로 파편이 인접한 곳에서 양측의 경계 의례·본능을 동시에 증폭시켰고, 평소라면 조우만 하고 지나갔을 상황이 전투로 비화했다.

**해소 계기**: site_wolfpine_trail에서 원정대가 이리솔 부족 팩 우두머리와 교전한 뒤, 이빨바람이 솔라룸측 단린에게 토템의 의미를 설명한다. 동시에 근처 심장로 파편이 양측의 적대감을 자극하고 있었음을 원정대가 발견한다. 첫 번째 '오해의 증거'가 확보된다.

### 솔라룸 vs 회상 결사 (`faction_solarum` × `faction_pale_conclave`)

| 항목 | 내용 |
| --- | --- |
| 표면 갈등 | 이단과 정화 |
| 숨겨진 진실 | 솔라룸이 과거 봉인 오용의 주범이며, 회상 결사는 그 정화 재판에서 살아남은 이단의 후예다 |
| encounter_link | `bastion_front`, `mark_execute` |
| first_resolution_window | `chapter_2_site_1` |
| 솔라룸→회상 찌르기 | "영원한 질서에 대한 모독. 정화 받지 못한 자들." 존재 자체를 이단으로 부정. 정화 재판이라는 제도적 폭력. 단린은 초반에 이 방향을 무의식적으로 공유하다가 ch2에서 자기 자신이 가해자임을 발견 |
| 회상→솔라룸 찌르기 | "네 신앙이 우리 조상을 죽였다. 네 정화 재판이 우리 기억을 살해했다." 정화 의식이 기억 소거와 동일한 원리라는 사실. 묵향은 기록·보관의 언어로 솔라룸의 은폐를 고발 |
| 금기어 | 회상 결사에게 "정화/소독/오염" (존재 소거 위협). 솔라룸에게 "기억 살해자/도굴꾼" (ch2 이전에는 자각 없음) |
| 증거 사물 | 기억 용기(memory vessel), 정화 재판 집행 기록, 단현 일지, 격자 파편 가루 |
| 양보 비용 | 솔라룸: 정화 재판 중단, 80년 오용의 공식 인정, 단린의 사제 정체성 일시 반납(회수 가능 hook — 휘장은 반납하되 새 신앙 형태 모색의 시작점). 회상 결사: 솔라룸에 대한 복수 포기, 공동 피해자로 재분류 수용 |

**구체적 오해 에피소드**: 솔라룸 사제 계급은 회상 결사의 존재 자체를 '영원한 질서'에 대한 모독으로 간주한다. 정화 재판에서 회상 결사 기억 용기(memory vessel)를 악마의 유물로 판정하고 파괴한다. 회상 결사 입장에서는 이것이 동료의 기억을 살해하는 것과 같다. 양측의 적대감은 신앙 vs 존재 보존이라는 근본적 가치 충돌에 기반한다.

**해소 계기**: site_sunken_bastion에서 원정대가 요새 지하의 기록 보관소를 발견한다. 솔라룸 학자들이 80년 전 봉인 격자 파편을 방어 장치로 전용한 기록이 나온다. 단린이 자신의 신앙이 은폐의 도구였음을 처음 인지하는 순간이다. 회상 결사의 묵향은 이 기록을 솔라룸의 죄를 묻는 게 아니라, 모두가 심장로에 속았음을 증명하는 증거로 사용한다. 단 정화 재판 자체는 심장로가 없었어도 일어났을 인간 권력 구조의 폭력이라는 점은 ADR-0024 결정 5번 가드레일대로 외부 메커니즘 탓으로 회피하지 않는다.

### 솔라룸 vs 그물 결사 (`faction_solarum` × `faction_lattice_order`)

| 항목 | 내용 |
| --- | --- |
| 표면 갈등 | 유적 침입자 탄압 |
| 숨겨진 진실 | 그물 결사는 솔라룸보다 먼저 이 땅을 지키고 있던 1800년 비밀 결사의 후예다 |
| encounter_link | `protect_carry`, `control_cleanse` |
| first_resolution_window | `chapter_4_site_1` |
| 솔라룸→그물 찌르기 | "우리 영토에서 나온 적대적 결사. 자기 정체를 1800년 숨겨온 자들." 영토 주권과 위협 판정. 방벽 기술이 무력화되는 공포. 단린에게는 "숭배하던 유물이 약탈품이었다"는 자기 파괴적 진실 |
| 그물→솔라룸 찌르기 | "너희가 우리의 격자를 뜯어 무기로 썼다. 우리의 신성한 매듭을 망치로 두드렸다." 격자 파편 약탈의 물리적 증거. 공한은 감정 없이 사실만 말하는데, 그 건조함이 더 무거움 |
| 금기어 | 솔라룸이 "우리 유물/신성한 유물"이라 부를 때 (약탈품을 신성화). 그물 결사에게 "침입자/유령 결사" (1800년 비밀 후예에게) |
| 증거 사물 | 격자 패턴이 새겨진 솔라룸 방어 장치, "신성 유물"(=약탈품), Aegis Sentinel이 만지자 반응하는 격자 |
| 양보 비용 | 솔라룸: 격자 파편 전량 반환, 방벽 기술 기반 상실, 군사적 열세 수용. 그물 결사: 솔라룸의 80년간 무지를 악의가 아닌 오해로 수용. 공한 개인: 격자 심부에 잠정 잔류(회수 가능 hook — 외전에서 격자 밖으로 잠시 투사 또는 다른 시간대 회신) |

**구체적 오해 에피소드**: 그물 결사가 각성하면서 격자 복원을 시도하자, 솔라룸 군은 이를 유적에서 나온 적대적 결사의 침공으로 판단한다. 특히 그물 결사의 장벽·정화 능력이 솔라룸의 방어 진지를 무력화하는 것처럼 보이기 때문에 위협 판정이 올라간다. 솔라룸은 자신들이 전용한 격자 파편이 그물 결사의 신성한 매듭이었다는 사실을 모른다.

**해소 계기**: site_glass_forest에서 솔라룸 군이 사용하는 방어 장치가 그물 결사의 격자 파편임이 물리적으로 증명된다. Aegis Sentinel이 장치를 만지자 격자 패턴이 반응하고, 단린은 자신이 숭배하던 '신성 유물'이 약탈품이었음을 받아들여야 한다. 솔라룸의 오용이 격자 풍화의 주범이었다는 마지막 퍼즐이 맞춰진다.

### 이리솔 부족 vs 회상 결사 (`faction_wolfpine_tribes` × `faction_pale_conclave`)

| 항목 | 내용 |
| --- | --- |
| 표면 갈등 | 사냥터 오염과 영역 분쟁 |
| 숨겨진 진실 | 둘 다 심장로 공명에 다른 방식으로 반응할 뿐이다 |
| encounter_link | `tempo_swarm`, `sustain_grind` |
| first_resolution_window | `chapter_3_site_1` |
| 이리솔→회상 찌르기 | "너희 때문에 사냥감이 썩는다." 묘역 주변 변이를 오염으로 귀결. 이빨바람은 냄새·질감으로 변이를 감지하고 본능적으로 원인을 회상 결사에게 돌림 |
| 회상→이리솔 찌르기 | "너희 사냥이 기억의 잔향을 짓밟는다." 기억 보존이라는 존재 이유가 위협받는 것. 묵향은 이리솔 부족의 무지를 탓하지만 악의로 보지는 않음 |
| 금기어 | 회상 결사에게 "썩은/오염된/시체" (격자 부산물 노출의 결과인 외양을 폄하). 이리솔 부족에게 "먹이/짐승" (지성 부정) |
| 증거 사물 | 변이된 사냥감 시체, 기억 용기 파편, 고대 공존 기억 투사에서 같은 물을 마시는 장면 |
| 양보 비용 | 이리솔 부족: 묘역 주변 사냥 자제, 영역 독점 포기. 회상 결사: 사냥터 경계 존중, 기억 잔향 확산 통제. 묵향 개인: 가장 오래된 기억을 정화 대가로 잠정 봉헌(회수 가능 hook — 동료들이 그녀가 잃은 기억의 단편을 보유, 후속 외전에서 들려줄 수 있음) |

**구체적 오해 에피소드**: 이리솔 부족은 묘역 주변에서 사냥감이 변이하거나 사라지는 것을 회상 결사의 오염 탓으로 본다. 회상 결사는 이리솔 부족의 사냥이 묘역 주변의 기억 잔향을 교란한다고 본다. 양측 모두 상대가 자신의 생존 기반을 훼손한다고 믿는다. 실제 원인은 격자 부산물이 두 결사·부족 모두에게 다른 방식으로 영향을 미치는 것.

**해소 계기**: site_ruined_crypts에서 기억 용기(memory vessel)가 깨지면서 고대의 기억이 투사된다. 그 기억 속에서 이리솔 부족과 회상 결사의 조상 — 옛 인간 갈래들 — 이 같은 공간에서 평화롭게 공존하던 장면이 나온다. 심장로 가동 이전의 세계. 이빨바람과 묵향이 동시에 그 기억을 목격하면서, 적대감이 인위적으로 만들어진 것임을 처음 의심한다.

### 이리솔 부족 vs 그물 결사 (`faction_wolfpine_tribes` × `faction_lattice_order`)

| 항목 | 내용 |
| --- | --- |
| 표면 갈등 | 본능적 적대 (이질 결사 첫 조우) |
| 숨겨진 진실 | 그물 결사 격자가 이리솔 부족의 의례·본능 공명을 완화할 수 있다 |
| encounter_link | `weakside_dive`, `control_cleanse` |
| first_resolution_window | `chapter_4_site_2` |
| 이리솔→그물 찌르기 | 본능적 위협 반응 — 격자 공명이 이리솔 부족의 사냥꾼·전사 의례를 자극. 이빨바람은 "몸이 먼저 반응한다. 머리보다 빨리"라고 표현. 이성으로 억누르지만 의례적 본능이 경고함 |
| 그물→이리솔 찌르기 | "너희 본능이 지금 느끼는 것이 원래 너희 것이 아니다." 이리솔 부족의 정체성(의례·생활 양식=자아)을 부정하는 효과. 공한은 사실을 말하지만 그 사실이 이리솔 부족에게는 존재 위협 |
| 금기어 | 이리솔 부족에게 "기계/부품/출력" (자연·문화 존재를 기술 대상으로 환원). 그물 결사에게 "유령 결사/허깨비" (1800년 비밀 후예의 존재 부정) |
| 증거 사물 | 격자 공명 주파수 변화, 의례 반응 전후의 토템 진동 차이, Prism Seeker의 주파수 조절 시연 |
| 양보 비용 | 이리솔 부족: 그물 결사의 존재 자체를 수용, 의례적 반응이 외부 자극에 영향받음을 인정(자아 위기). 그물 결사: 공명 출력 조절 약속, 이리솔 부족 영역에서의 격자 작업 사전 협의 |

**구체적 오해 에피소드**: 그물 결사가 각성하면서 방출하는 격자 공명이 이리솔 부족의 의례·생활 양식을 자극한다. 이리솔 부족은 이것을 포식자의 위협으로 해석하고 공격적으로 반응한다. 그물 결사는 격자 복원에 집중하느라 이리솔 부족의 반응을 이해하지 못한다.

**해소 계기**: site_starved_menagerie에서 Prism Seeker가 격자 공명의 주파수를 조절해 이리솔 부족의 의례적 반응을 완화하는 것에 성공한다. Trail Scout가 이를 목격하고, 그물 결사의 격자가 적대의 도구가 아니라 평화의 도구일 수 있음을 씨족에 보고한다. 이리솔 부족과 그물 결사 사이에 최초의 협력이 시작된다.

### 회상 결사 vs 그물 결사 (`faction_pale_conclave` × `faction_lattice_order`)

| 항목 | 내용 |
| --- | --- |
| 표면 갈등 | 유적 소유권 분쟁 |
| 숨겨진 진실 | 둘 다 격자·봉인 시스템의 후손이지만, 서로 다른 방식으로 그 시스템과 얽혀 있다 — 회상 결사는 격자 부산물 노출이 누적된 정화 재판 살아남은 자들의 후예, 그물 결사는 1800년 봉인 작업에 참여한 결사의 비밀 후예 |
| encounter_link | `summon_pressure`, `control_cleanse` |
| first_resolution_window | `chapter_3_site_2` |
| 회상→그물 찌르기 | "이 유적은 우리 기억의 집이다. 너희가 격자라 부르는 것 안에 우리 조상의 잔향이 잠들어 있다." 기억 소유권 주장. 묵향은 보관자의 권위로 이를 주장 |
| 그물→회상 찌르기 | "기억은 격자의 부산물이다. 격자가 먼저고, 기억은 나중이다." 회상 결사의 존재 이유를 부차적으로 환원. 공한은 의도 없이 사실을 말하지만 그 사실이 회상 결사에게는 존재론적 격하 |
| 금기어 | 회상 결사에게 "부산물/부차적/부수적" (기억의 가치 부정). 그물 결사에게 "침탈자/도둑" (봉인 비밀 결사에게 도둑이라 부르는 것) |
| 증거 사물 | 고대 봉인 작업에 회상 결사 조상이 간접 관여한 기억 기록, 격자와 기억 용기의 공명 패턴 일치, 유골 과수원 유골 뿌리 |
| 양보 비용 | 회상 결사: 유적 독점 포기, 공동 관리 수용. 그물 결사: "기억은 부수적"이라는 관점 철회, 기억 보존을 격자 복원과 동등한 목표로 승격 |

**구체적 오해 에피소드**: 회상 결사는 묘역과 유적이 자신들의 기억 저장소라고 주장한다. 그물 결사가 각성하면서 같은 유적의 격자를 복원하려 하자, 이를 기억 저장소에 대한 침탈로 간주한다. 그물 결사는 격자 복원이 우선이고, 회상 결사의 기억 보관은 격자 운영에 부수적이라고 본다.

**해소 계기**: site_bone_orchard에서 공한이 각성하면서 그물 결사의 비밀 기록 속에 저장된 고대 기억을 공유한다. 그 기억 속에서 최초의 봉인 작업에 회상 결사 조상이 — 그 시점엔 평범한 인간 갈래로 — 자발적으로 협력했음이 드러난다. 회상 결사와 그물 결사는 적이 아니라 같은 봉인 프로젝트의 서로 다른 후손이었다. 묵향과 공한의 공감이 이 관계의 전환점이 된다.

### Post-Midpoint Method Debate (ch4~5)

중반 이후 정화/봉인/파괴 논쟁이 각 세력 쌍의 갈등에 어떻게 매핑되는지를 정리한다.

#### 세 선택지의 비용-보상 구조

| 선택지 | 핵심 이점 | 핵심 비용 | 감정적 무게 |
| --- | --- | --- | --- |

## 작성 지침

- 세력 단독 lore는 `docs/02_design/narrative/world-building-bible.md`를 복제하지 않는다.
- gameplay link는 encounter family 혹은 answer lane ID로 연결한다.
- chapter resolution 표현은 동일한 어휘 세트를 유지한다: `hidden`, `seed`, `escalate`, `teaser`, `partial resolve`, `confrontation`, `negotiate`, `mutual understanding`, `deepen empathy`, `mostly resolved`, `aligned`, `stabilized`, `resolved with cost-and-recovery`, `resolved with loss-and-recovery`, `stabilized with absence-and-recovery`.
- 옛 종족 외양 어휘는 본 문서에서 사용하지 않는다 (구체 어휘 목록은 ADR-0024 retcon 금지 규칙 및 `world-building-bible.md` 명명 규칙 참조). 4 인간 세력의 결사·문화 어휘로만 표현한다.
- "되돌릴 수 없는" 영구 상실 표현은 회피한다. ADR-0024 결정 2번에 따라 "잠정"·"회수 가능 hook"을 함께 명시한다.
- 정화 재판 같은 인간 권력 폭력은 "심장로가 없었어도 일어났을" 인간 본성 어둠임을 명시한다 (ADR-0024 결정 5번).
