VII · MOTION

Motion · Notes Only

애니메이션 없음 · 텍스트 가이드
M-01
Snap Modal
모달·드로어·toast 모두 200ms · cubic-bezier(.2,.7,.2,1) snap. opacity + 4~8px translate만. shadow trail · spring · bounce 금지. 닫힘은 들어올 때와 동일 시간, 동일 ease-out.
M-02
Dialogue Reveal
글자가 28ms/char 정도로 한 자씩 노출. 일본어 보이스 cadence에 sync — voice 끝나면 reveal도 종료. 스킵 시 즉시 전체 노출 + 다음 prompt.
M-03
Telegraph
공격 예고는 단순한 ring 또는 beam. archetype family color로 깜박임 1Hz, 3펄스 후 발동. 파티클·잔상·스파크 없음 — silhouette만 읽히면 충분.
