# permanent augment progression

- 상태: draft
- 최종수정일: 2026-03-29
- phase: prototype

## 목적

이 문서는 permanent augment progression 문법을 정의한다.

## 핵심 규칙

- permanent augment는 run 밖에서 유지된다.
- MVP 슬롯 수는 1개다.
- temporary augment의 최초 선택은 관련 permanent augment 후보 해금으로 이어질 수 있다.
- permanent augment는 희소하고 방향성을 가진 선택이어야 한다.

## MVP 구현 범위

- slot 1개
- 후보 해금 규칙
- expedition 종료 시 후보 제시 또는 진척 지급

## 장기 규칙

- multi-slot
- branching progression
- category mastery
- account-level unlock web
