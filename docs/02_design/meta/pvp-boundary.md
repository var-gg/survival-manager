# pvp boundary

- 상태: draft
- 최종수정일: 2026-03-29
- phase: prototype

## 목적

이 문서는 PVP의 범위를 MVP와 분리해 정의한다.

## 핵심 규칙

- PVP는 비-MVP다.
- MVP에서는 PVP를 구현하지 않는다.
- PVP가 나중에 도입되더라도 run 중 temporary augment는 적용하지 않는다.
- PVP에는 permanent augment만 적용한다.

## 이유

- run 중 temporary augment까지 포함하면 밸런스와 판별성이 급격히 무너진다.
- PVP는 run 내 운 요소보다 장기 progression과 roster 준비를 더 직접 반영해야 한다.

## 장기 규칙

- 별도 matchmaking
- PVP 전용 보상
- PVP 밸런스 보정
- async defense/offense 모델 가능성
