# Operator First Playable Checklist

## 반드시 먼저 실행할 메뉴 1개

- `SM/Bootstrap/Prepare Observer Playable`

- 상태: active
- 최종수정일: 2026-03-29
- phase: prototype

## 첫 전투까지 클릭 순서

1. Unity 열기
2. `SM/Bootstrap/Prepare Observer Playable`
3. `Boot.unity` Play
4. Town에서 `Quick Battle`
5. Battle replay 종료 후 `Continue`
6. Reward에서 카드 1장 선택
7. `Return Town`

## 확인 항목

- Town에서 recruit 카드 3개와 Quick Battle 버튼이 보인다.
- Battle에서 primitive actor / HP / 로그 / pause / speed / progress가 보인다.
- Reward에서 카드 선택 즉시 summary/status가 갱신된다.
- Town 복귀 후 save state가 남아 있다.
